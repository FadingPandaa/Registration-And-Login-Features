Registration and Login Features (Java) – Full Project Evolution

Project Overview

This project is a Java-based Registration and Login System developed in
three progressive parts. Each part represents an improvement in
functionality, structure, and overall system design.

The project demonstrates: - Object-Oriented Programming (OOP) - User
authentication - Data handling using collections - GUI interaction using
Swing (JOptionPane) - Incremental software development

------------------------------------------------------------------------

📚 Project Breakdown

🔹 Part 1: Initial Development

Part 1 represents the foundation of the project.

Features included: - Basic program structure - Simple user
input/output - Early implementation of registration and login logic -
Minimal validation

Purpose: - To demonstrate understanding of core Java concepts - To
establish the base functionality of the system

------------------------------------------------------------------------

🔹 Part 2: Pre-Modification Version

Part 2 builds on Part 1 with improvements and refinements.

Enhancements included: - Improved login and registration logic - Better
input validation - More structured code using classes - Introduction of
separate components (Login, SignUp)

Purpose: - To improve code organization and readability - To introduce
modular programming concepts

------------------------------------------------------------------------

🔹 Part 3: Final Version (Complete System)

Part 3 is the fully developed and improved version of the system.

Final features: - Complete registration and login system - Credential
storage using HashMap - GUI interaction using JOptionPane - Integration
with EasyKanban module - Unit testing included - Cleaner, modular, and
maintainable code

Purpose: - To deliver a complete and functional application - To
demonstrate applied knowledge of Java development

▶️ How to Run the Project

Option 1: NetBeans 1. Open NetBeans IDE 2. Click File → Open Project 3.
Select the project folder 4. Click Run

Option 2: Command Line 1. Navigate to the src folder 2. Compile: javac
registrationandloginfeatures/*.java 3. Run: java
registrationandloginfeatures.RegistrationAndLoginFeatures

------------------------------------------------------------------------

🧪 Testing

Unit tests are included in the test folder and can be executed using
NetBeans or JUnit.

------------------------------------------------------------------------

🔄 System Flow

1.  User selects:
    -   Login
    -   Sign Up
    -   Exit
2.  New users register and credentials are stored in memory
3.  Returning users log in using stored credentials
4.  Successful login leads to the EasyKanban module
